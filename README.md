# The-Frequency

A digital sanctuary built by Prodigal Robe.

## Vision

The Frequency is intended to feel like a functioning TV network rather than a playlist. It should:

- feel like live TV channels
- cycle channels every 6–12 hours
- pull top videos for each channel/category
- shuffle intelligently so the experience feels dynamic, not chaotic
- maintain device synchronization so every viewer sees the same content at the same time

## Goals

- Build as its own standalone app
- Use GitHub for version control
- Deploy through Vercel
- Embed or link the deployed app from Squarespace
- Lean mobile-first with a polished phone experience
- Avoid shorts and inconsistent profile thumbnails
